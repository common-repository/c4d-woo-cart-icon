Popup Cart, Side Cart, Sticky Add To Cart Button, Call For Price Button

== Description ==

One of boost sales plugin from [Coffee4Dev Team](http://coffee4dev.com).

Plugin create popup cart or side cart to increase revenue as well as profitability for your website, stores.

### Live Demo
- Click to see [live demo](http://30tet.coffee4dev.com/ "live demo") of C4D Woo Cart

### FEATURES
- Ajax Popup Cart
- Ajax Side Cart
- Call For Price for out of stock or empty price on single page
- Sticky Add to Cart button on single page

### PREMIUM FEATURE IN C4D Woocomemrce Ajax Popup Cart, Side Cart=
- Update quantity in popup cart
- Call For Price on archive page/ other
- Custom call for price text
- Side Cart on left side.
- Responsive Mode

### INSTALLATION & CONFIG

1. Upload `c4d-woo-cart-icon` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

### OPTION PAGE

To enable admin options please install [C4D Plugin Manager plugin](https://wordpress.org/plugins/c4d-plugin-manager/)
